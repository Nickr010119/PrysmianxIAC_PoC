# Portal de Diagnóstico & Competencias — Prysmian Group & IAC

Una landing page premium, minimalista y completamente responsiva diseñada en colaboración entre **Prysmian Group** e **IAC**. Este portal digital centraliza el acceso a las evaluaciones de diagnóstico y competencias mediante códigos QR interactivos y enlaces cortos optimizados, garantizando una excelente usabilidad desde cualquier dispositivo (móvil, tablet u ordenador).

## 🚀 Características Clave

- **Alineación de Marca:** Integración sofisticada de las paletas de colores corporativas y logotipos de ambas marcas.
- **Diseño Premium Minimalista:** Fondo dinámico con efectos de resplandor (*glows*) y componentes basados en el estilo *glassmorphism* (esmerilado).
- **Interactividad Intuitiva:** 
  - **Lightbox para Códigos QR:** Haz clic en cualquier código QR para ampliarlo en una ventana modal emergente de alta resolución para facilitar el escaneo directo desde dispositivos móviles.
  - **Copiar al Portapapeles:** Botones de copia rápida para los enlaces recortados con notificaciones visuales emergentes (*toast notifications*).
- **100% Responsivo:** Adaptabilidad perfecta desde pantallas móviles pequeñas hasta grandes monitores de escritorio.
- **Estándares Modernos:** Construido con HTML5 semántico, CSS3 moderno (utilizando variables nativas y microanimaciones de carga suaves) y JavaScript nativo.

## 🛠️ Tecnologías Utilizadas

- **HTML5:** Estructura semántica avanzada.
- **CSS3:** Estilos avanzados, variables de diseño corporativas, animaciones personalizadas y responsive design fluido.
- **JavaScript (ES6):** Funcionalidades dinámicas de interacción (copiado de enlaces, lightbox modal y avisos flotantes).
- **Google Fonts:** Fuentes tipográficas modernas (`Inter` y `Outfit`).
- **FontAwesome:** Iconografía premium.

## 💻 Ejecución Local

Para visualizar y probar la landing page en tu computadora localmente, no necesitas compilar nada ni instalar dependencias pesadas:

1. Clona este repositorio:
   ```bash
   git clone https://github.com/Nickr010119/PrysmianxIAC_PoC.git
   ```
2. Entra en la carpeta del proyecto:
   ```bash
   cd PrysmianxIAC_PoC
   ```
3. Abre el archivo `index.html` directamente en tu navegador preferido.
